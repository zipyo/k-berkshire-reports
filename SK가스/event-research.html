2014 M&A/구조조정 동부발전당진 인수 → 당진에코파워로 사명 변경, 1,160MW 석탄화력발전 사업권 확보 해당없음(10년 밖)
2016 자본배분 SK어드밴스드 지분 일부 매각(PIC에 약 1억 달러 유치) 확인 불가
2017 규제/리스크 제8차 전력수급기본계획(탈석탄) 발표로 당진에코파워 석탄발전 사업 무산 위기 확인 불가
2018 M&A/구조조정 당진에코파워 관련 1,091억원 손상차손 반영(토지 218억 + 건설중인자산 873억), 순손실 1,118억원 기록 확인 불가
2018 M&A/구조조정 2013년 분사한 윤활유 탱크터미널 사업부문 '지허브' 재합병 확인 불가
2018.12 경영진/지배구조 윤병석 대표이사 사장 선임 확인 불가
2019.3 규제/리스크 LPG 연료 사용제한 전면 폐지 → 수송용 LPG 차량 일반 판매 허용 해당없음(산업 전반 이벤트)
2019.10~11 경영진/지배구조 SK디스커버리, SK가스 지분 공개매수(약 91.4만주, 지분율 55.7%→약 65.7%) 확인 불가
2020 M&A/구조조정 사우디 PDH·PP·IPA 사업(APOC) 참여 결정, 1.03억 달러(약 1,436억원) 자본 집행 확인 불가
2020.7 사업 전략 당진에코파워 석탄발전 사업 공식 전환 → 태양광 9.8MW + ESS 24.5MWh 친환경 발전소로 상업가동 개시 확인 불가
2021.3 경영진/지배구조 SK디스커버리, SK가스 2차 공개매수(46.2만주, 지분율 67.2%→72.2%, 주당 109,000원) D+0 +10.88% 급등(10만5천원 돌파)
2022.3 자본배분/사업 전략 울산GPS 착공(1.4조원 투자, 세계 최초 GW급 LPG-LNG Dual Fuel 복합발전소, 1,212MW) 확인 불가
2022 R&D/기술 롯데케미칼·에어리퀴드코리아와 롯데SK에너루트 합작 설립(지분 45%/45%/10%) — 수소연료전지 발전 확인 불가
2022.12 고객/공급망 한국동서발전과 1조 693억원 규모 CEC LNG 터미널 TUA 장기계약(~2045.6) 체결 확인 불가
2022.12 자본배분 CEC 울산 LNG 탱크 건설 착공(총 투자 2,428억원, 2026.12 완공) 확인 불가
2023 사업 전략 미국 텍사스 ESS 사업 진출(SK이터닉스·Apex Clean Energy와 'SA Grid Solutions' 합작, 2029년까지 900MW 목표) 확인 불가
2023 사업 전략 울산 수소복합단지 2.2조원 투자 결정(14만㎡, 2025년까지 추출수소·연료전지·액화수소 순차 건설) 확인 불가
2024.11 사업 전략 코리아에너지터미널(KET) 준공(총 64.5만kl LNG 탱크 3기) — 한국석유공사 51%/SK가스 45%/S-Oil 4% 구조 확인 불가
2024.12 사업 전략 울산GPS 상업운전 개시 (LPG-LNG 복합화력 1,212MW, 총 투자 1조 29억원) — SK가스 LNG 밸류체인 핵심 완성 확인 불가
2025.3 고객/공급망 S-OIL과 5,316억원 규모 LNG 저장탱크 TUA 장기계약(2026.7~2041.6) 체결 확인 불가
2025 경쟁/시장 LPG 내수 M/S 하락(38.4%→34.7%→33.5%, 2023~2025) — 국내 LPG 총수요 10.4% 감소 영향 해당없음
2025.4 R&D/기술 SA Grid Solutions 텍사스 100MW ESS 첫 프로젝트 상업가동 개시 확인 불가
2025.6 R&D/기술 롯데SK에너루트 울산하이드로젠파워 2호(20MW) 수소연료전지 발전소 상업운전 개시 확인 불가
2025 규제/리스크 LPG 수입업 저장시설 보유 요건 30일분 → 15일분 완화 — 진입장벽 일부 약화 해당없음
2026.2 M&A/구조조정 SK그룹 신재생 사업 재편 패키지 딜(약 1.8조원) — KKR이 SK이터닉스 우선협상대상자 선정, SK디스커버리 지분 30.98% 매각 대상 포함 해당없음(SK이터닉스 기준 +22% 강세)
2026.4.2 M&A/구조조정 사우디 APOC(PDH) 지분 15% 전량 매각 — Put option 행사로 1,956억원(1.29억 달러) 회수 확인 불가
2026.4.6 자본배분/M&A 울산GPS 지분 49% → 스틱·한투PE 컨소시엄에 1조 2,242억원 매각 (경영권 51% 유지, 미래성장 투자재원 확보) 확인 불가
2024~2026 자본배분 3개년 주주환원 정책(FY2024~2026): 연결지배주주순이익(경상)의 25% 이상 배당, ROE 12% 초과 시 자사주 매입·소각 추가 환원 검토. DPS 추이: 5,100원(2021) → 6,500(2022) → 8,000(2023) → 8,000(2024) → 9,000(2025, +12.5%) 해당없음(패턴 정책)
