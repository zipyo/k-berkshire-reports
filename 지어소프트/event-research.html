연도 카테고리 이벤트 시장 반응
2011 사업 전략 김영준 대표의 지어소프트 인수, 자회사 우리네트웍스(현 오아시스) 설립 → 유통·이커머스 사업 진출 해당없음 (참고용)
2013 사업 전략 우리생협과 위탁판매대리점 계약, 오프라인 직영 매장 개설 확인 불가
2018 사업 전략 오아시스마켓 온라인 새벽배송 플랫폼 런칭 (생산자 직거래 기반) → 사업구조 대전환점 확인 불가
2019 사업 전략 오아시스 매출 급성장 (연결 매출 1,851억원). 주가 2019년 연간 +30.23% 연간 +30%
2020 고객/공급망 코로나19로 새벽배송 수요 폭증, 지어소프트 영업이익 흑자전환(약 100억원). 주가 연간 +235.71% 연간 +236%
2021.07 자본배분(외부 투자) 오아시스, 유니슨캐피탈로부터 500억원 투자 유치 — 기업가치 7,500억원 평가. 지어소프트 지분가치 단순환산 약 5,000억원 주가 연간 +49.65%(3월 기준)
2022.01 사업 전략/자본배분 2차전지 소재 자회사 '지어솔루션' 설립 (100% 출자, 자본금 100억원) + 250억 유상증자 참여. 니켈도금강판 생산 → LG에너지솔루션·삼성SDI 타깃 확인 불가
2022.03 사업 전략 지어솔루션 대구국가산단 1,530억원 투자 계획 발표 (1차 830억 + 2025년까지 추가 700억), 402명 신규고용 확인 불가
2022.12 M&A/구조조정 오아시스 코스닥 상장 예비심사 통과. 케이뱅크와 플랫폼 연동 MOU 체결 확인 불가
2023.02 M&A/구조조정 오아시스 IPO 자진철회 — 수요예측 희망밴드 3.05만~3.95만원 대비 2만원대 주문 몰리며 공모가 미달. 구주매출 620억원 확보 기회 소실 D+1 시간외 -10%, 정규장 -15% 급락
2023.05 고객/공급망 오아시스마켓-케이뱅크 멤버십 서비스 론칭 (디지털금융-이커머스 최초 결합). 공동 금융서비스·제휴상품·ESG 협업 계획 확인 불가
2023 사업 전략 오프라인 직영점 조정 (3월 대치2호점, 5월 잠실점 폐점) 확인 불가
2024.05 자본배분 100억원 규모 자사주 취득 공시 (128.2만주, 5/13~8/9). 주주가치 제고 목적 확인 불가
2024.07 M&A/구조조정 11번가 인수 추진 — 지어소프트 주식 + 자회사 루트 신주를 11번가 지분 100%와 교환 제안. FI 반대로 무산 확인 불가
2024.12 자본배분 기업가치 제고 계획(밸류업) 공시 — 자사주 소각 포함한 주주환원 계획 발표 확인 불가
2024~2025 사업 전략 오아시스 언양 물류센터 구축, 경상/충청권 새벽배송 권역 확장 해당없음
2025.02 자본배분 추가 100억원 자사주 취득 + 전량 소각 결정 (밸류업 계획 이행) 확인 불가
2025.04 M&A/구조조정 오아시스마켓, 티몬 최종 인수예정자로 선정 (116억원, 100% 신주인수 방식). 안준형 오아시스마켓 대표가 티몬 신임 대표 겸임 확인 불가
2025.06 M&A/구조조정 서울회생법원, 티몬 회생계획안 강제인가 → 오아시스의 티몬 인수 확정. 추가 운영자금 약 65억원(미지급 임금·퇴직금) 투입 확인 불가
2025.10 자본배분 제8회 무보증 사모 전환사채(100억원, 표면이자율 0%) 전액 상환 완료 (만기 2025.10.08). 파생금융부채 212,874천원 → 0원 해당없음
2025 R&D/기술 R&D 비용 3년 연속 급증 (0.14% → 0.38% → 0.79%). AI 무인결제·디지털 셀프 체크아웃 특허 확보 해당없음
