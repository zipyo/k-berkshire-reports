2017 규제/리스크 세무조사 법인세 추징금 약 157억원 부과 → 해당 연도 당기순손실 약 206억원 기록 확인 불가
2018.03 경영진/지배구조 각자대표 체제 변경: 이완진·배건우 → 이완진·이영섭 각자대표. 이영섭 전 대표는 2015년 입사, 경영지원본부장 출신 전문경영인 확인 불가
2018.07 규제/리스크 발사르탄 발암물질 검출 파동: 중국산 원료에서 NDMA 검출. 대한뉴팜 '엔피포지정' 5/80mg·5/160mg 판매중지·회수. 업계 전반적 이슈였으나 대한뉴팜도 해당 품목 매출 타격 확인 불가
2020.01 경영진/지배구조 이영섭 대표이사 사직, 이완진 단독대표 체제 복귀. 회사 측 "개인사유로 인한 사직" 설명 확인 불가
2020.12 M&A/구조조정 Baverstock GmbH 청산 → Caspian Sunrise Plc 지분 직접 보유로 전환. 2007년 260억원에 투자한 카자흐스탄 유전개발 투자구조 단순화. 관계기업투자처분손실 261억원 일시 인식. 영업이익 248억원에도 당기순손실 99억원 기록. Caspian Sunrise 지분 10.8%(약 2.25억주) 공정가치 약 75억원으로, 원투자액(~330억원) 대비 대규모 가치 훼손 확인 불가
2021~2025 사업 전략 인체의약품 중심 체질 개선: 이원석 2세 경영 전후로 인체의약품 매출 비중·R&D 비중 확대, 동물의약품 비중 축소 추세. 매출 1,666억(2021) → 2,042억(2023)으로 성장. 2021년부터 18분기 연속 흑자 해당없음
2022.05 사업 전략 Caspian Sunrise 유전 생산능력 2배 증가 발표: 카자흐스탄 BNG 광구 일일 생산량 1,400배럴 추가 D+1 +41%
2023.01 경영진/지배구조 이원석 대표이사 사장 취임 — 2세 경영 본격화. 이완진 회장 일신상 사유로 대표이사 사임(회장직 유지). 이원석(1977년생, 이완진 장남) 단독 대표이사 사장 승진. 미래사업본부 신설, "경청·실천·책임" 경영 방침 발표 확인 불가
2024.04 경영진/지배구조 이원석 대표 블록딜(시간외매매)로 10만주 추가 취득: 지분율 6.97%(100만주) → 7.66%(110만주). 매도 측은 이완진 회장 딸 이지민 실질 지배 특수관계법인(아벤트코리아·엠앤비솔류션즈). 승계 잠재 변수 차단 + 지분율 강화 효과 확인 불가
2024.05~2025.08 자본배분 향남 GMP 신공장 건설: 2024.05 기공식(429억원 규모 발표) → 2025.08 준공(실제 총 투자 약 810억원). 연면적 15,150㎡, 지하1층·지상5층. 연간 정제 8억정, 캡슐 2억개 생산 가능. 동결건조주사제 역량 확보. 2027.01 본격 가동 목표. 건설중인자산 787억원(2025년말). 장기차입금 188억→540억 급증, 순차입금 279억→770억, 자본조달비율 18%→35%로 레버리지 대폭 확대 준공 시 시장 무반응(시총 1000억 붕괴)
2024.09 R&D/기술 DH푸졸정 출시: 세계 최초 반려견용 플루코나졸 제제. 2019년 개발 시작, 인체용 '푸졸정' 기반 동물용 전환 개발. 반려동물 의약품 시장 진입 본격화 해당없음
2024.12 R&D/기술 비만 4주 지속형 주사제 공동개발 계약: 대웅제약·티온랩·다림바이오텍과 체결. 세마글루타이드 기반, 마이크로스피어 약물전달 플랫폼 활용. 약물 로딩률 20%(업계 최고 수준 표방). 비임상 진행 중, 국내 임상 1상 신청 예정 해당없음
2016~2025 자본배분 배당 정책 (패턴): DPS 80원(2021) → 100원(2022~2025) 유지. 배당성향 약 7~10% 수준. 유상증자·자사주 매입 이력 없음. 신공장 투자에 자본 집중 배분 해당없음
2022~2024 R&D/기술 개량신약·제네릭 파이프라인 지속 출시: 테넬틴정·테넬포민서방정(당뇨, 2022), 라피듀오정(소화기 개량신약, 2024.03), 다파포민서방정(당뇨 제네릭, 2023) 등. 비만용제 파이프라인 다수 제제연구~품목허가 준비 단계 해당없음
2022~2024 경쟁/시장 중소 제약사 수익성 악화 환경: 제네릭 약가 인하 압력 지속, CSO 경쟁 심화. 대한뉴팜 영업이익률 16.0%(2021) → 13.9%(2022) → 9.2%(2023~2024) → 5.8%(2025)로 지속 하락. 동종 중소 제약사(하나제약·삼일제약·삼천당제약 등)도 유사한 수익성 하락 추세 해당없음
