1. 디지털 Scuttlebutt 요약

정보원 핵심 발견 체크리스트 연관
경쟁사 Anti-VEGF 주사제 과점 시장(Eylea/Vabysmo/Lucentis)에서 경구용 치료제는 전 세계적으로 미개척. 경쟁사 Roche는 경구용 vacasinabin을 Phase 2 후 중단(2026.01), Ocuphire APX3330은 Phase 2 완료 후 파트너 모색 중, Kalvista·Rezolute는 초기 임상. 경구 망막치료제 개발의 어려움과 동시에 기회를 반증. CU01 당뇨병성 신증 분야도 신장 섬유화 직접 억제제 경쟁 부재 Q1, Q4, Q11
고객 Anti-VEGF 불응 비율 ~40%, 진단 후 치료율 40% 미만 → 경구제 미충족 수요 큼. CU06 Phase 2a에서 경구제 중 세계 최초 시력개선 입증(BCVA 고용량군 +5.8자), CU01 Phase 2b에서 uACR 21~22% 유의미 개선. 환자 접근성(주사→경구)이 시장 확대 핵심 동인 Q1, Q4, Q5
직원 잡플래닛 평점 2.7/5(6건). 긍정: 수평적 문화, 좋은 연봉, 능력있는 동료, 워라밸. 부정: 꼰대 문화, 체계 부재, 의사소통 부족. 리뷰 수 극소(6건)로 대표성 제한적. 연구인력 20명(박사 12명)의 소규모 조직 Q7, Q8, Q9
공급업체 CMO 기반 위탁생산(자체 생산설비 없음). 대성팜텍 합병으로 원료의약품 내재화 — API 수입·유통 역량 확보, 5년 내 매출 140억 전망. 글로벌 CRO 시네오스 헬스와 Phase 2b 계약 체결 → 글로벌 임상 인프라 확보 Q4, Q10, Q11
업계 전문가 Rivasterat Phase 2a 결과 ASRS Late-Breaking Abstract 채택(2024.07), MT-103 ARVO 2026 구두발표 수락 — 학술적 인정. BIO USA 2025에서 미국 VC 포함 40건 미팅. 보령과 CU01 전략적 제휴(MOU). 복수 국내 제약사와 CU01 기술이전 논의 진행 중 Q1, Q3, Q9
기술/연구 권영근 의장: SCI 논문 225편, 특허 25건, 한국혈관학회 회장 역임. 특허 포트폴리오 등록 39건/출원 38건, Evergreening 전략으로 2045년까지 보호. MT-103 전임상에서 Eylea·Vabysmo 대체항체 대비 우수한 Tie2 활성화·VEGF 억제·혈관 투과성 감소 입증. SOLVADYS® + EAGLES® 이중 플랫폼 구축 Q3, Q11

교차 검증 불일치 사항

  • 잡플래닛 직원 리뷰 vs 경영진 구조: 직원 리뷰에서 "꼰대 문화"·"체계 부재" 지적이 있으나, 경영진 구조(학술 창업자 + J&J 출신 전문경영인)는 전문적으로 설계됨. 소규모 조직 특유의 구조 미비가 "꼰대 문화"로 체감되었을 가능성 — 다만 리뷰 표본이 6건으로 판단 유보
  • 경구 망막치료제 경쟁 구도: Roche는 vacasinabin Phase 2 후 중단, Ocuphire는 파트너 모색 중 → 경쟁사 탈락은 시장 진입 장벽의 높이를 보여주지만, 동시에 큐라클의 Phase 2a 성공이 더 차별화됨을 방증

Scuttlebutt 한계

  • 잡플래닛 리뷰 6건으로 직원 정서 대표성 부족 (소규모 기업 특성)
  • 프리커머셜 바이오텍으로 고객(환자)·공급업체 관점의 상업적 평가 불가 — 임상 데이터로 대체
  • 기술이전 논의 상대방(글로벌 제약사) 비공개로 BD 진척 교차검증 제한적
  • 온라인 정보 대부분이 기업 IR 또는 보도자료 기반 — 독립적 비판적 분석 소스 부족

2. 피셔 15가지 체크리스트

A. 성장 동력 (Q1-Q4)

Q2→Q3→Q4 = "성장 엔진의 세 기어"

문항 평가 검증 근거
Q1: 향후 매출 성장 가능성 △ ★★★ TAM: 망막질환 글로벌 $339B(2031), 만성신부전 5조원, 알츠하이머 $245B(2033) — 세 시장 모두 대형. 차별화: 경구용 망막치료제·신장섬유화 억제제 모두 경쟁제품 없는 First-in-Class. 임상 증거: CU06 Phase 2a 세계최초 경구 시력개선, CU01 Phase 2b 유의미 효능. 그러나 현재 매출 사실상 제로(2025년 7백만원), 상업화까지 최소 5~7년 소요. 기술이전 수익은 CU01이 2026~2027년 가시화 가능성 있으나 확정 아님. 성장 잠재력은 압도적이나 시현 시점의 불확실성 존재
Q2: 신제품/신사업 개발 의지 ✓ ★★★ Rivasterat 플랫폼에서 DME→AMD→면역항암→궤양성대장염→동물의약품까지 적응증 5개 확장. CU01(신증)→CU71(알츠하이머)로 ED Blocker 기전 확대. EAGLES® 항체 플랫폼(MT 시리즈 8개) 별도 구축. CAIT 설립(2024)으로 차세대 후보물질 발굴 전담. 원료의약품 사업(대성팜텍)으로 사업 다각화. 파이프라인 18개 이상(전임상 포함)으로 다중 shot-on-goal 전략 명확
Q3: R&D 효율성 ✓ ★★★ 연 107.5억 R&D 투자, 연구인력 20명(박사 12명)의 소규모 팀으로 높은 연구 생산성: CU06·CU01 각각 Phase 2b 도달, MT-103 전임상에서 Eylea/Vabysmo 대비 우수성 입증. 학술 성과: ASRS Late-Breaking, ARVO 2026 구두발표, 권영근 의장 SCI 225편. 국가신약개발사업 2건 선정(MT-101, MT-103). R&D 전액 당기비용 처리(보수적 회계). 맵틱스 공동R&D로 비용 분담. 대학 원천기술 License-in 모델로 초기 R&D 비용 최소화
Q4: 영업/마케팅/유통 역량 △ ★★☆ 유재현 대표의 J&J/얀센 22년 BD 경력이 핵심 자산. BIO USA 2025에서 40건 미팅, 보령 MOU 체결, 복수 제약사와 CU01 기술이전 논의 중. 그러나 떼아 계약해지 전례(파트너사 사유이긴 하나 결과적 매출 절벽). 상업화 제품 부재로 영업/마케팅 실행 역량 미검증. 원료의약품 사업부 신설(2025.03)로 유통 채널 확보 시작 단계. L/O 모델 특성상 자체 영업 조직보다 BD 역량이 중요하며, 이 부분은 구축 중

C. 조직/경영 역량 (Q7-Q9)

문항 평가 검증 근거

6. 투자 포인트 (3개)

  1. 경구용 망막질환 치료제 First-in-Class 지위: Rivasterat은 Phase 2a에서 경구제 중 세계 최초 시력개선을 입증했고, 경쟁사 Roche는 경구 후보물질을 중단. 경구 제형이라는 구조적 차별화(주사 대비 환자 접근성·편의성)는 Anti-VEGF 불응환자(40%)·미치료환자(60%)라는 대규모 미충족 수요에 직접 대응하며, TAM $339B(2031) 시장에서 보완적 포지셔닝 가능.

  2. 이중 임상 성공 + 다중 파이프라인: CU06(망막, Phase 2a 성공)과 CU01(신증, Phase 2b 성공)의 임상 데이터가 모두 유의미한 결과를 보여 단일 파이프라인 리스크를 분산. MT-103(아일리아/바비스모 대비 우수 전임상), CU71(알츠하이머 BBB Stabilizer) 등 차세대 파이프라인도 과학적 근거 축적 중. 2045년까지 특허 보호로 장기 진입장벽 확보.

  3. 위기 대응에서 입증된 경영진 실행력: 떼아 계약해지(2024.05)라는 사업 위기 후 ①권리 전부 무상 복귀, ②자체 Phase 2b 전환 결정, ③FDA Type C 미팅 완료, ④글로벌 CRO 계약, ⑤유상증자·대성팜텍 합병으로 재무 안정화, ⑥보령 MOU + 복수 제약사 L/O 논의까지 18개월 내에 체계적으로 실행. 학술 창업자(권영근)+글로벌 BD 전문가(유재현)의 상호보완적 리더십이 작동.

7. 리스크 (3개)

  1. 현금 소진과 추가 자금조달 압박: 현금+유동금융자산 약 240억(2025말), 연간 영업현금유출 ~134억 → 런웨이 약 1.5~2년. CU06 Phase 2b 임상비용이 본격 투입되면 소진 가속. 기술이전(CU01 등) 미성사 시 2027~2028년 추가 유상증자·CB 발행 불가피 → 36% 이상의 추가 주주 희석 가능성. 바이오텍 자금조달 환경 악화 시 런웨이 리스크 심화.

  2. 임상 실패의 이산적(binary) 리스크: CU06 Phase 2b(2026 IND 신청→2028 결과 예상)가 핵심 가치동인이나, Phase 2a→2b 성공 전환율은 업계 평균 30~40%. CU01은 Phase 2b 성공했으나 Phase 3 설계·비용·파트너 확보가 과제. 전임상 파이프라인(MT 시리즈, CU71)은 임상 진입까지 2~3년 추가 소요. 하나의 파이프라인이라도 핵심 임상에서 실패하면 기업가치의 비선형적 하락 불가피.

  3. 기술이전 의존 모델의 구조적 취약성: 수익화 전략이 글로벌 제약사 L/O에 전적으로 의존. 떼아 계약해지(2024) 전례가 보여주듯, 임상 성공에도 파트너사 사정에 의한 해지 가능. 현재 체결된 L/O 계약 없음. CDA 단계 논의가 계약으로 전환되지 않을 리스크 상존. 자체 상업화 인프라(생산·영업)가 전무하여 L/O 실패 시 대안 부재.
