사업 구조

경쟁 구도

  • 시장 규모(TAM):
  • EAI(Enterprise Application Integration) 글로벌: 약 $150~170억 (2024~2025), CAGR 12%+ (2025~2033 전망). 아태지역 CAGR 약 14.8%로 가장 빠른 성장
  • ECM/문서중앙화 글로벌: 약 $592억 (2023) → $1,312억 (2028), CAGR 14.2%
  • 국내 EAI/ESB: 별도 정량 데이터 미확인. 금융·제조·공공 중심 안정적 수요

  • 국내 문서중앙화: 랜섬웨어 사고 연평균 96% 증가 추세 속 수요 확대. 2025 상반기 침해사고 신고 전년 대비 ~15% 증가

  • 주요 경쟁사:

  • 연계솔루션(EAI/ESB): 인스피언 (2024 코스닥 상장, SAP EAI 컨설팅 특화), 인센트(INZENT, ESB/EAI 통합 연계), 글로벌 — IBM WebSphere, TIBCO, Software AG webMethods, MuleSoft
  • 보안솔루션(문서중앙화): 사이버다임 (국내 시장점유율 1위, 1,200+ 고객사, 2024년 크레센도에쿼티로부터 1,000억원 투자 유치), 넷아이디, 이스트소프트

  • APM/클라우드솔루션: Dynatrace (글로벌 APM 1위, 모코엠시스가 리셀), Netscout

  • 시장점유율: 모코엠시스는 자사를 "국내 최대 데이터 연계(EAI/ESB/OpenAPI) 전문 기업"으로 표현. 국내 EAI/ESB 분야에서 20년 이상 사업 영위하며 금융·제조·공공 분야 레퍼런스 다수 보유. 정량적 점유율 수치는 미확인(검색 결과 부족)

  • 경쟁 구도 특성: 국내 EAI/ESB 시장은 높은 전환비용으로 인해 기존 솔루션 교체가 어렵고, 소수 전문 업체 중심의 과점 구조. 문서중앙화 시장은 사이버다임이 선두이나 공공 부문(행안부 G드라이브)에서 모코엠시스가 강한 레퍼런스 보유. 인스피언의 2024년 코스닥 상장으로 경쟁 가시화(비교기업으로 모코엠시스 선정됨)

  • 윤영호 (공동 대표이사): 2025.1월 선임. 구체적 이력 미확인(검색 결과 부족)
  • 진충열 (부사장, 보안사업 총괄): "기업의 핵심 과제는 방대한 데이터 속에서 필요한 정보를 찾지 못해 업무가 중단되는 것"이라며 AI 기반 지능형 문서중앙화 비전 제시

  • (전임) 조천희 (전략사업부문 대표, 2021~2024), 고재현 (솔루션사업부문 대표, 2021~2024)

  • 대표이사 변경 이력:

  • 2021.03: 각자대표 체제 전환 — 조천희/고재현 공동 대표, 김인수 이사회 의장

  • 2025.01: 김인수/윤영호 공동 대표로 변경

  • 보상 구조: 구체적 금액 미확인(검색 결과 부족). 상장 전후 주식매입선택권(스톡옵션) 부여 이력 존재 — 주식보상비 2022~2023년 각 3억원 인식, 이후 행사 진행(2024년 4억, 2025년 1억 행사)

기술/IP

  • 핵심 기술:
  • MI(MIDAS Integration) 시리즈: EAI/ESB/MCI/MCA/FEP/Open API 등 연계솔루션 풀 라인업. MI Advanced가 최신 버전으로 클라우드·API 기술 적용
  • Mpower EZis-C: 로컬 저장 금지 + 가상 네트워크 드라이브 기반 문서중앙화. 분산파일시스템 기반 대규모 운영 기술력 (행안부 19개 부처 10만 명 운영 실적)

  • MOAI(MOcomsys AI): BERT·RAG·GPT-4 기반 AI 솔루션. 문서 검색/챗봇/유사문서 추천

  • 특허/IP: 보안솔루션 관련 15건, 연계솔루션 관련 15건 (프로그램저작권 + GS인증 + 기술인증). 2024~2025년 신규 등록 활발 — MoAI 저작권(2024.07), MI Advanced V10 저작권+GS인증(2024.08~09), MFT V1.0 저작권+GS인증(2024.12~2025.01)

  • R&D 투자: 매출 대비 6.4~7.2% 수준 유지 (제7기 20.3억원). 전액 판관비 처리(자산화 없음). 주요 R&D — MI Advanced for Cloud(MSA 재설계, 2025.03~2026.03), MFT 솔루션 고도화, AI 유사문서 추천(BERT 파인튜닝)

  • 기술 대체 리스크: 클라우드 네이티브 전환 시 EAI/ESB 수요가 API 게이트웨이/마이크로서비스로 이동할 가능성 존재. 이에 대응하여 MI Advanced for Cloud(MSA 구조)와 MI API Gateway 개발 진행 중. IBM 라이선스 의존도는 고객 계약에 인상분 전가 가능 구조로 리스크 제한적
